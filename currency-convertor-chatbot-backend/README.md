#  Currency Converter Chatbot

A conversational AI chatbot that performs *real-time currency conversions* using *Google Dialogflow* and a *Python Flask webhook*. The bot can interpret natural language, fetch live exchange rates from an API, and reply in messaging platforms like Telegram or web chat.

---

##  Features

-  Converts currencies such as *USD, INR, EUR, GBP, JPY, AUD, CAD*
-  Supports *natural language queries*
-  Handles *small talk* and *fallback responses*
-  Deployed backend using *Render*
-  Integrated with *Telegram Bot* for real-time usage
-  Clean and interactive chatbot experience

---

##  Tech Stack

-  Natural Language Understanding - **Dialogflow**
-  Backend API - **Flask (Python)**
-  Deployment - **Render**
-  External Data Source - **ExchangeRate-API**
-  Chat Platform - **Telegram**
